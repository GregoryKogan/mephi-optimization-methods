# Универсальный шаблон для написания документов и конспектов

## Поддерживаемые языки верстки

- LaTeX.
- MarkDown.

## До начала использования

Вся настройка данного способа писать документы находится в конфигурационном файле *build-conf.toml*. 

## Использование

Для конвертирования исходников, которые написаны на одном из поддерживаемых языков верстки, в один PDF файл достаточно в конфигурационном файле *build-conf.toml* сделать следующие изменения:

- Написать список источников, которые будут входить в PDF, в поле *src*.
- Написать имя выходящего файла в поле *out-name*. Формат выходящего файла важен.
